scene-db

Search and extract scenes from autonomous driving and robotics log data

GitHub Quick Start
$ pip install scene-db

Features

📥

Dataset Ingestion

Parse KITTI raw data (oxts, images, point clouds) and split into time-based scene chunks automatically.

🔍

Scene Search

Search scenes by text queries on auto-generated captions describing vehicle motion and behavior.

📊

Feature Extraction

Compute speed, distance traveled, and generate rule-based captions for each scene chunk.

📤

Scene Export

Export specific scenes with all associated files (images, LiDAR, IMU) to a clean directory structure.

🗄

SQLite Storage

Zero-dependency local storage. No servers, no setup. Everything in a single file.

⌘

Simple CLI

Four commands: ingest, index, search, export. That's it.

Quick Start

$ scene-db ingest /data/kitti/2011_09_26_drive_0001_sync
Ingesting /data/kitti/2011_09_26_drive_0001_sync ...
Done. Created 24 scene chunks.

$ scene-db search "moving forward"
Found 12 scene(s):

  [kitti_2011_09_26_drive_0001_sync_003]
    vehicle moving forward, 42 km/h, traveled 58.3 m
    frames 30-39, 2011-09-26T13:02:40 - 2011-09-26T13:02:44

$ scene-db export --id kitti_2011_09_26_drive_0001_sync_003 -o ./my_scene
Exported 50 files to ./my_scene
